# Amazon Sales Data Analysis (SQL + Python)

##  Project Overview
This project analyzes Amazon e-commerce sales data using Python and SQL. The dataset was sourced from Kaggle and contains information on orders, products, customers, shipping details, and payments.

The goal is to extract business insights such as revenue trends, top-performing product categories, and customer behavior.

---

##  Tools Used
- Python (Pandas)
- SQLite
- SQL
- VS Code

---

##  Project Workflow
1. Loaded Amazon sales dataset (CSV) using Python
2. Stored data into SQLite database
3. Performed SQL-based analysis using queries
4. Extracted business insights from the data

---

##  Key Analyses Performed
- Top-selling product categories by quantity
- Revenue analysis by category
- Top customers based on spending
- Sales distribution by state
- Order status distribution

---

##  Key Insights
- Identified best-performing product categories
- Analyzed revenue contribution across categories
- Evaluated customer purchasing behavior
- Reviewed order fulfillment trends

---

##  Skills Demonstrated
- Data cleaning and preparation
- SQL querying (JOINs, GROUP BY, aggregations)
- Data analysis using Python
- Business insight generation
